# JobVision Navigation Calc

اکستنشن کروم برای محاسبه زمان رسیدن به محل شغل از موقعیت شما.

## نصب

1. فایل `generate-icons.html` را در مرورگر باز کنید و آیکون‌ها را ذخیره کنید
2. پوشه `icons/` را با آیکون‌های تولید شده پر کنید
3. در Chrome به `chrome://extensions/` بروید
4. "Developer mode" را فعال کنید
5. "Load unpacked" را بزنید و پوشه پروژه را انتخاب کنید

## استفاده

1. روی آیکون اکستنشن کلیک کنید
2. مختصات عرض و طول جغرافیایی خود را وارد کنید (یا "موقعیت فعلی" را بزنید)
3. ذخیره کنید
4. به سایت jobvision.ir بروید و لیست مشاغل را جستجو کنید
5. زمان رسیدن به هر شغل کنار نام خیابان نمایش داده می‌شود
